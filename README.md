## 🏁 Getting Started

### 📋 Requirements

```
Node >= 18.17.1

yarn >= 1.22.19
```

### 💫 Precondition

환경 변수 설정
root 경로에 있는 .env.sample 파일을 복사하여 .env로 수정해 주세요.<br/>
아래 내용을 참고하여, 프로젝트 실행 전에 필요한 항목에 값을 할당해주면 됩니다.

```
# env.sample

VITE_NAVER_MAP_API_KEY=
VITE_KAKAO_MAP_API_KEY=
```

### ⏳ Installation

1. install dependencies for packages

```
yarn install
```

2. start local project

```
yarn dev
```
